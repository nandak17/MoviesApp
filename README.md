## Getting Started

1. **Clone the repo**  
    ```
    git clone https://github.com/nandak17/MoviesApp.git
    ```
2. **Open in Android Studio**  
    Recommended: Android Studio Electric Eel or newer  
3. **Build & Run**  
    Run on emulator or physical device with developer mode enabled
